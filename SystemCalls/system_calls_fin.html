System Call Reference

fork() - Creates a new process by duplicating the calling process. After fork(), both processes execute the same program, but with different PIDs.

exec() - Replaces the current process image with a new program. This system call is often used after fork() to run a different program in the child process.

open() - Opens a file and returns a file descriptor that can be used in subsequent operations like read() and write().

read() - Reads data from an open file descriptor into a buffer, returning the number of bytes read.

write() - Writes data from a buffer to an open file descriptor, returning the number of bytes written.

close() - Closes a file descriptor, freeing the resources associated with it.

wait() - Suspends execution of the calling process until one of its child processes terminates.

exit() - Terminates the calling process and returns a status value to the parent process.
