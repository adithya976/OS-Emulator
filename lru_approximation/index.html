Second-Chance (LRU Approximation) Simulator

This simulator demonstrates the Second-Chance (Clock) algorithm, which approximates LRU using a FIFO queue and a reference bit. Each page gets a "second chance" if its bit is 1. The algorithm rotates the queue until a page with bit 0 is found and replaced.
